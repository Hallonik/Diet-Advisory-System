Overview
The Diet Advisory System is a web-based application designed to provide personalized dietary recommendations based on user-specific information such as age, weight, height, and dietary preferences. By leveraging machine learning algorithms, the system aims to promote healthier eating habits and assist users in achieving their nutritional goals.

Features
Personalized Diet Plans: Generate diet plans tailored to individual user profiles, considering factors like age, weight, height, and dietary restrictions.
Nutritional Analysis: Provide detailed nutritional information for recommended foods, helping users understand their dietary intake.
Health Goal Tracking: Assist users in setting and monitoring health goals, such as weight loss, muscle gain, or maintaining a balanced diet.
Interactive User Interface: Offer an intuitive and user-friendly interface for seamless interaction and navigation.
